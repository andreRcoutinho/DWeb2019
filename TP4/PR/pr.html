Abstract

Este projeto consistiu numa transformação XSL aplicada ao dataset dos arqueossítios, gerando um documento XML para cada um. Através de outro documento XSL, é possível associar uma stylesheet a cada ficheiro XML gerado, resultando em páginas HTML. Foi também criado um index para ligar a todos os ficheiros gerados. Tudo isto é servido através de um servidor em node.js.

Deliverables

  • XSLT que apresenta cada arqueossítio
  • XSLT que separa cada arqueossítio
  • XSLT que gera o índice
  • Index.html
  • Servidor em node.js
